Full-Stack Web Developer

A front-end web developer at Nucleus Global, a world-wide medical communications company, where I am responsible for the creation, development, and implementation of internal and external websites as well as web solutions for mobile devices with a specific focus on large scale JavaScript projects.

I've completed training to become a full-stack web developer through Rutgers Coding Bootcamp, a rigorous yet exciting program where I learned in-depth web development including AngularJS, APIs, NodeJS, MySQL, Express, and so much more.

Winner of the V12 Group Hackathon. In seven hours my team and I built a web application that allowed new and existing businesses to analyze the demographics of a geographical region to best parlay their next business decision.

When I'm not working, I feed my love for traveling. I've snowboarded through the evergreen mountains of Vermont and canoed the Bioluminescent Bay of Puerto Rico. I've backpacked from Costa Rica to Nicaragua and have enjoyed the sushi in Japan. I've seen with my own eyes the Crown of Thorns at Notre Dame in Paris as well as the Holy Grail at the Cathedral of Valencia in Spain. I've swam the beautiful beaches of the Philippines and climbed the Lanikai Pillboxes of Hawaii to catch the Golden Sunrise Sky. I have much more to see...

Projects

Trip Chats

A full-stack project that allows users to post itineraries of trips they plan on going to. Other users can post comments to suggest to the original poster so they can get advice of what places to see and activities to do. This gives users access to insight from locals or other travelers to make the best of their trip.

  • MySQL stores the data
  • Express is the web framework
  • AngularJS is the router and controller
  • Node.js is the engine
  • Bootstrap is the CSS framework

My To Do List

This AngularJS app allows the user to add items to a to-do list and check it off as they complete each item. The user can also delete items off the list completely. Since this is an Angular app it allows for data-binding. Try using this app and notice how the list changes as the you interact with it without the browser needing to reload the page or link to another page.

RU Review

There are plenty of places to eat at Rutgers! RU Review is a place where users can read reviews about the many food spots on and around campus. Users can create accounts and write their own reviews as well as update and delete. RU Review is a back-end CRUD project utilizing many web tools:

  • Materialize is the CSS framework
  • Node.js is the engine
  • npm packages:
    • Express
    • Express-Session
    • Sequelize
    • Passport
    • Passport-Local

    • Handlebars
    • Dotenv
  • MySQL stores the data
