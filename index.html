Daily Briefing

  • jQuery is the engine

RU Review

There are plenty of places to eat at Rutgers! RU Review is a place where users can read reviews about the many food spots on and around campus. Users can create accounts and write their own reviews as well as update and delete. RU Review is a back-end CRUD project utilizing many web tools:

  • Materialize is the CSS framework
  • Node.js is the engine
  • npm packages:
    • Express
    • Express-Session
    • Sequelize
    • Passport
    • Passport-Local
    • Body-Parser
    • Express-Handlebars
    • Bcryptjs
    • MySQL
    • Handlebars
    • Dotenv
  • MySQL stores the data
